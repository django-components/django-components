# Plugins

A catalog of community plugins and extensions for django-components is on the way.

In the meantime, you can discover packages that build on django-components by
searching [PyPI](https://pypi.org/search/?q=django-components) or browsing the
[GitHub discussions](https://github.com/django-components/django-components/discussions).

Built something worth listing here?
[Open an issue](https://github.com/django-components/django-components/issues) and
let us know.
